افزونه Smart Media Manager یک راهکار سبک، ماژولار و بسیار پیشرفته برای سیستم مدیریت رسانه وردپرس است. این سیستم با هدف بهینه‌سازی دیسک سرور، افزایش سرعت لود صفحات با تزریق لایت‌باکس‌های تاچ و مسدودسازی هوشمندانه ساخت تصاویر بندانگشتی زائد طراحی شده است.

✨ قابلیت‌های کلیدی و عمیق

📐
مدیریت ابعاد پیشرفته (ISQM) جلوگیری از اسراف فضا بر روی هارد دیسک سرور با مهار کدهای فرعی قالب‌ها که به صورت خودکار انواع سایز اضافی ایجاد می‌کنند.
📸
تزریق بومی فریم‌ورک Fancybox اسکن خودکار کانتینرهای گالری پیش‌فرض وبلاگ، ساختارهای المنتور و گوتنبرگ و تبدیل آن‌ها به اسلایدر تمام صفحه با پشتیبانی از رفتارهای تاچ موبایل.
🎯
مکانیزم هوشمند و بدون خطای Fallback اگر عکسی توسط مدیر سایت حذف شده یا سایز درخواستی آن در هاست موجود نباشد، افزونه به صورت خودکار نسخه اصلی را جایگزین می‌کند تا از نمایش آیکون تصویر شکسته (404) در سایت ملوگیری شود.
💎
مدیریت فشرده‌سازی تا مرز ۱۰۰٪ کنترل کامل خروجی تصاویر با کیفیت نهایی بدون افت پیکسل برای فایل‌های با فرمت JPEG و PNG.

📖 مستندات فنی و راهنمای توسعه‌دهندگان

این افزونه با رعایت اصول توسعه هسته وردپرس پیاده‌سازی شده و از تداخل با ساختارهای ادمین جلوگیری می‌کند:

۱. فیلترهای لایه مهار اندازه و فشرده‌سازی

افزونه از هوک‌های زیر برای مدیریت کیفیت تصاویر استفاده می‌کند که می‌توانید کدهای خود را به آن‌ها متصل کنید:

// مهار فشرده‌سازی پیش‌فرض هسته وردپرس
add_filter('jpeg_quality', function() { return 100; });
add_filter('wp_editor_set_quality', function() { return 100; });

۲. معماری تابع ضد شکستگی تصاویر (Fallback Mechanism)

هوک image_downsize به شکلی شرط‌بندی شده است که آرایه‌ای شامل آدرس تصویر اصلی و ابعاد آن را برگرداند. این کار مانع از کرش یا ایجاد حلقه‌های بی‌پایان در مروگر می‌شود.

🛠️ نحوه نصب و راه‌اندازی

۱. پوشه فشرده افزونه را در مسیر wp-content/plugins/ آپلود کنید.

۲. افزونه را از بخش مدیریت افزونه‌ها در پیشخوان وردپرس فعال نمایید.

۳. گزینه‌ها را از زیرمنوی جدید رسانه ← Smart Media Manager مدیریت کنید.

Smart Media Manager is a modern enterprise-grade optimization toolkit designed for the WordPress Media Library architecture. It features aggressive server storage savings, unified high-performance lightbox assets orchestration, and automatic broken image container handling.

✨ Advanced Deep Feature Set

📐
Intelligent Size Quota Management (ISQM) Prevents server disk storage exhaustion by blocking redundant background thumbnail layouts generated by active themes.
📸
Automated Global Fancybox Hooks Scans and wraps legacy core layouts, Gutenberg galleries, and Elementor media modules into dynamic touch-friendly lightboxes.
🎯
Asynchronous Runtime Source Fallback Interceptors listen to media request failures on front-end rendering engines and replace missing dimensions with the master resolution automatically.

📖 Technical Integration Guide

Built completely on top of non-destructive native filter architectures keeping administrative panels decoupled from functional layout shifts:

1. Pixel-Perfect Compression Handlers

The system hooks seamlessly into WordPress internal filters to override compression rules:

// Enforce maximum lossless rendering matrix overrides
add_filter('jpeg_quality', function() { return 100; });

2. Safe Context Evaluation

The image_downsize logic contains context safety locks that immediately abort execution when is_admin() is true to protect database attachment assets metadata during editing operations.
